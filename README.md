# Seminar\_assignment\_git



What the project does

Creates an R package which imports and filters RNA-seq counts.

Derives Differentially expressed genes based on log2 fold change and False discovery rate. Further performing an enrichement analysis on the DEGs. The results are then exported as an excel table and plotted. 



Why the project is useful

It is a mini-software package built for an assignment. It allows reproducibility of results and can be used on different datasets. 



How users can get started with the project

Download the count table and sample table of your data or create these files from your raw data. Download the package and run the analysis.
